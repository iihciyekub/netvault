Platform Overview

NetVault is an internal research collaboration platform for managing reference PDFs through both a command-line interface (CLI) and a web interface. Users can upload local reference collections to the cloud without manually preparing or maintaining the original PDF metadata. The platform extracts identifiers such as DOIs, while its automated processing pipeline handles metadata cross-checking, file validation, normalization, and ingestion. Authorized team members can later locate literature with a single command or a web search and retrieve selected files through high-concurrency downloads, streamlining document archiving, sharing, and collaborative use.

Usage Policy

NetVault is intended exclusively for authorized members of the team and must not be made publicly accessible. Every PDF made available through the platform must be lawfully obtained by an individual user from a legitimate academic institution or authorized scholarly source for personal academic and research use. Bulk redistribution, forwarding, resale, commercial exploitation, automated mass downloading, and any unlawful or unlicensed distribution are strictly prohibited. Users must comply with the applicable publisher licences, copyright requirements, and institutional access and subscription terms governing each document, and any downloaded PDF must be deleted within 24 hours after use. NetVault provides internal document-management and collaboration capabilities only; it does not grant or transfer copyright, access rights, or redistribution rights. Each user remains solely responsible for the legality and compliance of their actions and document use.
