Step 01

Install / Update

Install the lightweight local CLI, then keep it current from GitHub.

curl -fsSL https://raw.githubusercontent.com/iihciyekub/netvault/main/scripts/install.sh | bash
nv update
nv --version
Step 02

Login

Use the shared NetVault server account shown below. The token is saved locally for future commands.

nv login https://iiaide.com/nv --username polyu --password '!1@2#3Qwe'
nv status
nv logout
Step 03

Get PDFs

Download by one DOI, many DOI values, or a text file. Downloads run in parallel and resume automatically.

nv download 10.1016/j.ijpe.2018.04.006 --to ./downloads
nv download 10.1016/j.ijpe.2018.04.006 10.1234/example --to ./downloads
nv download --file ./dois.txt --to ./downloads
Step 04

Upload PDFs

Upload one PDF, multiple files, or a folder. Existing files are skipped by SHA-256.

nv upload ./paper.pdf
nv upload ./paper-a.pdf ./paper-b.pdf
nv upload ./papers
nv upload ./paper.pdf --doi 10.1234/example
